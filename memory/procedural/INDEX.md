## Creating New Workflows

1. **Identify repetition** - Notice when you do the same thing multiple times
2. **Document steps** - Write down what you actually do
3. **Validate** - Follow your own documentation
4. **Refine** - Update based on experience
5. **Share** - Make available to team

---

## Maintenance

### After using a workflow
- Note any steps that were unclear
- Add missing prerequisites
- Update time estimates

### Monthly review
- Archive unused workflows
- Update outdated steps
- Consolidate similar workflows
