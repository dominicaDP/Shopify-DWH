# Ideas

This folder stores ideas captured via `/idea` command.

## Structure

- `product-ideas.md` - Product and feature ideas
- `tech-ideas.md` - Technical improvements and tools
- `business-ideas.md` - Business and process ideas
- `process-ideas.md` - Workflow improvements

## Idea Format

```markdown
### [Idea Title]

**Captured:** [date]
**Category:** [product / tech / business / process]
**Priority:** [HIGH / MEDIUM / LOW]
**Status:** [new / exploring / building / archived]

**Description:**
[What is the idea?]

**Problem it solves:**
[Why does this matter?]

**Next steps:**
- [ ] [Action item]

**Related:**
- [Link to related project or pattern]
```

## Idea Lifecycle

1. **Captured** - Raw idea via `/idea`
2. **Categorized** - Auto-sorted by type
3. **Evaluated** - Priority assigned
4. **Actioned** - Moved to project tasks or archived
